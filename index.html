Arcade

Available Games:

Tic-Tac-Toe

The object of Tic-Tac-Toe is to get three in a row. The first player is known as X and the second is O. Players alternate placing Xs and Os on the game board until either opponent has three in a row or all nine squares are filled, resulting in a draw.

Snake

The object of Snake is to grow as large as you can by eating apples without running into yourself or any walls. Use the provided movement keys to direct the snake around the game board to chase the apple. The game ends when the player crosses themselves or hits a wall.
